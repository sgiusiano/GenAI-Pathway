# ChatGPT Prompt Engineering for Developers

In this directory, you'll find Jupyter notebooks with exercises from the [ChatGPT Prompt Engineering for Developers](https://learn.deeplearning.ai/courses/chatgpt-prompt-eng) course, using updated versions of the OpenAI libraries.

## Prepare environment

- Copy the .env.example file to .env and fill the values

```bash
copy .env.example .env
```

- Create a virtual environment

```bash
python -m venv venv
```

- Activate the virtual environment

```bash
venv\Scripts\activate
```

- Install the requirements

```bash
pip install -r requirements.txt
```
